Google Maps Demo

Usage of Google Maps Platform API to build a simple demo showcasing the following features:
  1. Map interface
  2. Custom Markers
  3. Custom Polylines
  4. Pan & Zoom Smooth Animations
  5. Reset Map to Original State Function
